1. INSTALL FIRST
   You must run 'npm install' before starting development.
   This downloads all required dependencies.

2. EMAIL SETUP
   The contact form requires EmailJS setup.
   Get credentials from emailjs.com and update Contact.jsx

3. UPDATE CONTENT
   Replace all placeholder text with your actual information.
   This makes the portfolio personalized to you.

4. TEST RESPONSIVENESS
   Test on mobile, tablet, and desktop before deploying.
   Use Chrome DevTools to test different screen sizes.

5. BUILD BEFORE DEPLOY
   Always run 'npm run build' before deploying.
   Only the 'dist' folder needs to be deployed.
